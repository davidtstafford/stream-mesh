Gang Wars: Stream Mesh Game Guide

Welcome to Gang Wars! This is a chat-driven game where you can join gangs, earn currency, buy weapons, and battle other players or gangs. Use the commands below in chat to play.

Getting Started

  • Register: ~gw register
  • Create a gang: ~gw creategang <name>
  • Join a gang: ~gw joingang <gang>
  • Leave your gang: ~gw leavegang

Currency & Banking

  • Deposit: ~gw deposit <amount>
  • Withdraw: ~gw withdraw <amount>

Weapons & Inventory

  • Buy weapon: ~gw buy <weapon>
  • Upgrade weapon: ~gw upgrade <weapon>
  • Shop: ~gw shop (Lists all weapons and prices)

Combat

  • Attack player: ~gw attack <player>
  • Attack gang: ~gw attackgang <yourGang> <targetGang>

Leaderboards

  • Leaderboard: ~gw leaderboard (Top 3 players, top 3 gangs, your rank)

Admin Commands (Super Mods Only)

  • Reset game: ~gw adminreset
  • Give currency: ~gw admingive <player> <amount>

Tips & Info

  • Currency name and game status can be set by the broadcaster in the Games tab.
  • Major events are announced in chat and via TTS overlays.
  • Check the Games tab in the Stream Mesh app for leaderboards and more info.

Example Gameplay

~gw register
~gw creategang TheLegends
~gw joingang TheLegends
~gw deposit 100
~gw buy pistol
~gw attack rivalPlayer

Have fun and may the best gang win!
