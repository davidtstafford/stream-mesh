Stream Mesh Chat Commands

Below are the available chat commands for Stream Mesh. Use these in your Twitch chat to control TTS and other features.
Note: Some commands require moderator or super moderator permissions.

Command Description Permission
~voicesShows a link to the full list of available TTS voices.Viewer
~setvoice [voice]Set your personal TTS voice. Example: ~setvoice BrianViewer
~myvoiceShow your current TTS voice setting.Viewer
~viewervoice @userShow another user's current TTS voice.Viewer
~setviewervoice @user [voice]Set another user's TTS voice. Example: ~setviewervoice @user BrianSuper Mod
~enabletts on|offEnable or disable global TTS for the channel.Super Mod
~enablevoice @user on|offEnable or disable TTS for a specific user.Mod
~setperm @user viewer|mod|smodSet a user's permission level.Super Mod
~helloReplies with a hello message.Viewer
~commandsShows this commands list link.Viewer

For a full list of TTS voices, use ~voices or visit voices.html.
