Amazon Polly Setup for StreamMesh TTS

StreamMesh supports Amazon Polly for high-quality Text-to-Speech (TTS) voices. This guide explains how to set up your own AWS account, create credentials, and enable Polly voices in the app.

1. Create an AWS Account

  • Go to aws.amazon.com and sign up for a free account if you don't have one.

2. Create an IAM User for Polly

  1. Visit the AWS IAM Console.
  2. Click Users in the sidebar, then click Create user.
  3. Enter a username (e.g., streammesh-tts).
  4. Click Next.
  5. Under Set permissions, choose Attach policies directly.
  6. Search for and select the policy: AmazonPollyReadOnlyAccess (recommended) or AmazonPollyFullAccess (if you need more permissions).
  7. Click Next and then Create user.
  8. On the new user's summary page, scroll to the Access keys section.
  9. Click Create access key.
  10. When prompted for the use case, select Application running outside AWS.
  11. Click through the prompts. When the key is created, you will see the Access Key ID and Secret Access Key. Copy or download these nowâ€”this is the only time you will see the secret key.

3. Enable Polly in StreamMesh

  1. Open StreamMesh and go to Admin > TTS.
  2. Find the Amazon Polly section.
  3. Enter your Access Key ID, Secret Access Key, and AWS Region (e.g., us-east-1).
  4. Click Save.

4. Select a Polly Voice

  • After saving your keys, the app will fetch available Polly voices.
  • Choose your preferred voice from the list.

5. Privacy & Security

  • Your AWS keys are stored securely on your device and never sent to StreamMesh servers.
  • You are responsible for your AWS usage and costs. See Polly pricing.

6. Troubleshooting

  • If voices do not appear, double-check your keys and region.
  • Make sure your IAM user has Polly permissions.
  • Check your AWS usage limits if you hit errors.
